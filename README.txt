Description :

	A Java Desktop Application which can be used in creating projects, assigning tasks to team members and track their the progress.


Background Image : 

	Background Image can be changed by specifying the absolute path of the image in path.txt file in 03 - Backgound Image directory.

	Use double slash in the path if you find difficulties in displaying the image ( E:\\SoftwareProjectManagementSystem\\03 - Background Image\\bg.png )

External Jars : 
	
	Copy all the files inside the 02 - External Jars directory and paste them in C:\Program Files\Java\jre\lib\ext 
	
	This location may vary based on the location of Java in your machine but regardless place inside jre\lib\ext
